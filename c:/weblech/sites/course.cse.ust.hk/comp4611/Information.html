    Scope

    The computer architecture defines the interface between computing hardware and software, and has a significant impact on the evolution of computer systems. This course studies the architectural design of modern computers. It covers topics including pipelining, memory hierarchy, input/output, instruction set design, vector processing, multiprocessing and "cloud" computers.
