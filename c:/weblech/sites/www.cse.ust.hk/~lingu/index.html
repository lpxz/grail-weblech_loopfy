Lin Gu is an assistant professor at the Department of Computer Science and Engineering. He received B.S. from Fudan University (1996), M.S. from Peking University (2001), and Ph.D. from the University of Virginia (2006). His current research interests include large-scale distributed systems and cloud computing, operating systems, wireless sensor networks, and energy-efficient computing.

Research projects

I have looked into quite a few things in computer systems, and below is a list of some projects. You are welcomed to visit our group web page for updated info about our recent endeavors.

  • System software for cloud computing
    The emerging cloud computing paradigm introduces new challenges to system software design. We design, implement and evaluat scalable file systems, flexible parallel processing support, and core OS services that run efficiently on a plurality of commodity compute nodes.

    Layer Zero (L0): A general-purpose substrate for cloud computing.

    DVM: A distributed virtual machine that comprises a large number of physical hosts.

    BIFS: A bit-interleaving file system that retains a certain level of compressibility while providing a strong privacy protection guarantee.

    CCMR: A cloud computing testbed used by multimple research groups for scientific computing and interdisciplinary research.

  • Virtk (a.k.a. t-kernel): OS design for general low-power systems
    A new OS kernel that supports virtual memory, preemptive scheduling, and reliable OS protection without traditional hardware support.

  • Wireless sensor networks

    Detection and classification sensornet: Lightweight detection and classification in wireless sensor networks

    Energy-efficient RFID inventory algorithms: Smart transmission power control algorithms can significantly reduce the energy consumption of RFID inventory operations without noticeably affecting the throughput of the system.

  • Energy efficiency and power management: Novel hardware, architecture and middleware solutions are needed to make wireless sensor network sustain a long period of time.

    Radio-triggered wakeup: A wireless-wakeup mechanism that can significantly increase the lifetime of a low-power wireless system.

  • Network protocols: TDMA based MAC and data-link layer protocols, low-latency routing protocols, and datacenter networking.
