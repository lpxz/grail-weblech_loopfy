Courses Taught

  • COMP660J: Hot Topics in Computer and Internet Communications
    This is a postgraduate course covering different aspects of dealing with congestion in computer networks. Notably, it covers congestion avoidance in virtual circuit based networks including the associated queueing theory aspects, as well as controlling congestion in datagram networks including the associated non-linear optimization aspects.
    Taught in: Spring 2006.
  • COMP660F: Quality of Service In the Internet
    This is a postgraduate course covering quality of service aspects of the Internet, including scheduling, policing, admission control, congestion control, active queue management, Integrated services an Differentiated services.
    Taught in: Fall 2000, Spring 2003.
  • COMP561: Computer Networks
    This is a postgraduate course intended for students without UG background in networking. It covers a mixed collections of topics from COMP361 and COMP362.
    Taught in: Fall 2006.
  • COMP361: Computer Networks I
    This is a basic course in computer networks for undergraduate students. It covers the basic functionality of the five layers of the network protocol stack, including, applications and socket programming, TCP and congestion control, IP and routing in the Internet, data link layer, error detection, media access control (ALoha, CSMA,...)
    Taught in: Fall 2005, Fall 2004, Summer 2007, Summer 2008, Summer 2009, Summer 2010.
  • COMP252: System Software
    This is an undergraduate course on operating systems covering all aspects of an OS including processes, threads, synchronization, virtual memory, caches, resource allocation, scheduling, input/output, ...
    Taught in: Spring 2004.
  • COMP362: Computer Networks II
    This is an advanced course in computer networks for undergraduate students. It covers advanced aspects of networking, including, applications and socket programming, advanced TCP flavors and congestion control mechanisms, Quality of Service and Multimedia networking, Multicast routing, Network Security, Wireless networks, Ad-hoc networks,
    Taught in: Fall 2001, Fall 2002, Fall 2006.
  • COMP180: Computer Organization
    This is an undergraduate course on computer organization. It has two parts: assembly language programming (using both MIPS and x86 assembly languages), and basic hardware component of a computer (ALU, Datapath, Control, Memory, Input/Output)
    Taught in: Spring 2001, Spring 2002, Spring 2005, Spring 2009, Spring 2010, Spring 2011.
